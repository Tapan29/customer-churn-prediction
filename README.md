# Bank Customer Churn Prediction

## Overview

This project predicts whether a bank customer is likely to churn using Machine Learning.

The model was developed using Scikit-Learn and deployed using Streamlit.

## Features

* Data preprocessing using Pipeline and ColumnTransformer
* Missing value handling
* One-Hot Encoding
* Feature Scaling
* Random Forest Classification
* ROC-AUC Evaluation
* Confusion Matrix
* Feature Importance Analysis
* Streamlit Deployment

## Dataset Features

* Customer Demographics
* Account Information
* Banking Activity
* Transaction Behavior
* Complaint History
